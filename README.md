A smart IoT-based weighing scale that measures body weight using a load cell and HX711 sensor, processes the data with an ESP32, and uploads it to ThingSpeak via Wi-Fi. The system analyzes weight trends using a 3-day moving average to estimate hydration status (Normal, Possible Dehydration, or Possible Water Retention). A responsive web dashboard displays real-time weight readings, historical trends, and hydration insights for easy health monitoring.
